漫展网 API 接口文档

v1.0.0

概述

漫展网（Cartoon）是基于 ThinkPHP 5 + FastAdmin 开发的漫画阅读平台，提供漫画浏览、章节阅读、用户管理、收藏关注、付费购买等功能。

Base URL

https://your-domain.com

数据格式

所有接口返回 JSON 格式，Content-Type: application/json

字符编码

UTF-8（数据库 utf8mb4）

认证方式

需要登录的接口需在请求中携带 Token，支持以下三种方式：

方式一：HTTP Header（推荐）

Token: {token_value}

方式二：Query 参数

?token={token_value}

方式三：Cookie

Cookie: token={token_value}

Token 有效期为 30 天（2,592,000 秒），可通过 刷新接口 续期。

响应格式

所有接口返回统一 JSON 结构：

// 成功响应
{
    "code": 1,
    "msg": "操作成功",
    "time": 1719876543,
    "data": { ... }
}

// 失败响应
{
    "code": 0,
    "msg": "错误信息",
    "time": 1719876543,
    "data": null
}

// 未登录（HTTP 401）
{
    "code": 0,
    "msg": "Please login first",
    "time": 1719876543,
    "data": null
}

// 无权限（HTTP 403）
{
    "code": 0,
    "msg": "You have no permission",
    "time": 1719876543,
    "data": null
}
